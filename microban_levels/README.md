These levels were taken from Microban by David Skinner (1997).
155 levels ordered easy to hard, all proven solvable with documented solutions.
Difficulty is ordered for humans, not algorithms.

Source: http://www.sneezingtiger.com/sokoban/levels/microbanText.html

Levels in this directory:

- level1.txt  ->  Microban Level 1    (easiest)
- level2.txt  ->  Microban Level 37
- level3.txt  ->  Microban Level 92
- level4.txt  ->  Microban Level 124
- level5.txt  ->  Microban Level 129
- level6.txt  ->  Microban Level 131  (this takes the longest - can take up to 329seconds)
- level7.txt  ->  Microban Level 155  (hardest - 'The Dungeon')
